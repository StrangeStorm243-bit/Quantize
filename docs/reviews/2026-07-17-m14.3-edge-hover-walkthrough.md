demo traffic. Trigger 3 did not fire.** On real datasets this is the number to re-measure; a rising
figure is capture-at-run *evidence* for a founder decision, never a cache authorization (§10.5).

## Anomalies / notes

- Playwright's center-point `hover()` was occluded for two edges (bbox centers under the palette /
  a node card at the default zoom); the walkthrough dispatched real bubbling `mouseover` events on
  the edge elements instead — the same DOM path a human hover produces (RF's own handler chain
  consumed them, as the served readouts prove). Human mouse hover on the visible path segments works
  normally at ordinary zoom (the fit-view case).
- One stray navigation click mid-walkthrough briefly selected the v1 run before the light-theme
  step — caught by the readout itself showing the honest 404 (step 10's light variant), which is
  the feature working as designed.
